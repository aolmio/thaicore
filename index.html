🟢 Live Syllables (คำเป็น)
Sound sustains freely — vocal pathway stays open.

Live if:
  • Ends with a long vowel, no final stop (e.g. มา maa)
  • Ends with a nasal: -n (น ณ ร ล ฬ ญ), -m (ม), -ng (ง)
  • Ends with a semi-vowel: -w (ว), -y (ย)
🔴 Dead Syllables (คำตาย)
Air column arrested abruptly — sharp pressure stop.

Dead if:
  • Ends with a short vowel, no final consonant (e.g. จะ ja)
  • Ends with an unreleased stop: -k (ก ข ค ฆ), -p (บ ป พ ฟ ภ), -t (ด ต ถ ท ธ ศ ษ ส…)

Memory Shortcuts: Mental models that map the logic of Thai tones to intuitive concepts.
🪞 The Mid Class Mirror Perfect Map
Mid class consonants are perfectly obedient. Whatever tone mark is written, that exact tone comes out. Ek → Low, Tho → Falling, Tri → High, Chattawa → Rising. No surprises.
🏴‍☠️ The Low Class Rebel Jump +1 Shift
Low class consonants with a tone mark shift the output one rung higher than expected. Mai Ek (normally → Low) on Low class → Falling. Mai Tho (normally → Falling) on Low class → High.
💥 Dead Syllable Pressure Spike Air Burst
Unmarked Low class + short dead stop (e.g. รัก) can't release air pressure — it spikes into High. Long vowel dead stop (e.g. พูด) has room to climb then drop → Falling. Like a balloon: tiny fast pop vs. slow arc.
🌄 High Class Natural Rise Default Up
Unmarked High class in live syllables (e.g. ขา) naturally produce a Rising tone. "High class = high ambitions" — they rise by default.
🔑 The ห Elevator Class Lift
ห (High class) silently elevates a following Low class consonant. หมา, หนา, หนัก all behave as High class even though ม, น are Low. ห provides the "elevator" — inaudible but class-changing.
🪨 Mid Class Default Anchor Ground Zero
Unmarked Mid in a live syllable = flat Mid tone. Unmarked Mid in a dead syllable = Low tone. Death drags it down one level.
