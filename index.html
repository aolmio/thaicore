🟢 Live Syllables (คำเป็น)
The syllable's sound can sustain freely — the vocal pathway stays open.

Signs of a Live syllable:
  • Ends with a long vowel and no final stop (e.g., มา maa)
  • Ends with a nasal: -n (น ณ ร ล ฬ ญ), -m (ม), -ng (ง)
  • Ends with a semi-vowel: -w (ว), -y (ย)
🔴 Dead Syllables (คำตาย)
The air column is arrested abruptly — pressure stops suddenly.

Signs of a Dead syllable:
  • Ends with a short vowel and no final consonant (e.g., จะ ja)
  • Ends with an unreleased stop: -k (ก ข ค ฆ), -p (บ ป พ ฟ ภ), -t (ด ต ถ ท ธ ศ ษ ส…)
🎵 The ห Leading Rule
When ห (high class) leads a low class consonant like ม, น, ง, ว, ล, ย, ร, the ห acts as a class elevator — the pair behaves as High class. This is why หมา (dog) is a Rising tone, not Mid.

Memory Shortcuts: These conceptual anchors map the underlying logic of Thai tones to intuitive mental models.
🪞 The Mid Class Mirror Perfect Map
Mid class consonants are perfectly obedient. Whatever tone mark is written, that exact tone comes out. Ek → Low, Tho → Falling, Tri → High, Chattawa → Rising. No surprises.
🏴‍☠️ The Low Class Rebel Jump +1 Shift
Low class consonants with a tone mark shift the output one rung higher than you'd expect. Mai Ek normally makes Low → but on Low class it makes Falling. Mai Tho normally makes Falling → but on Low class it makes High. Rebel jump!
💥 The Dead Syllable Pressure Spike Air Burst
Unmarked Low class + Short dead stop (e.g., รัก) can't release the air pressure — it spikes into High. A long vowel dead stop (e.g., พูด) has room to climb then drop: Falling. Think of a balloon — tiny fast pop vs. slow release arc.
🌄 High Class Natural Rise Default Up
Unmarked High class consonants in live syllables (e.g., ขา) naturally produce a Rising tone. Think "High class = high ambitions" — they rise by default, not fall.
🔑 The ห Elevator Class Lift
ห (high class) silently elevates the class of a following Low class consonant: หมา, หนา, หนัก all behave as High class even though ม, น are Low. ห provides the "elevator" — you don't hear it but it changes everything.
🪨 Mid Class Default Anchor Ground Level
Unmarked Mid class in a live syllable is your reference zero: flat Mid tone. Unmarked Mid class in a dead syllable drops to Low. Death drags it down.
